# 🏥 Cuidapp

## 📌 Descripción
 
Aplicación pensada para médicos de internación clínica que permite generar y enviar partes diarios de pacientes en formato PDF a sus familiares, facilitando la comunicación y seguimiento del estado clínico.

### 🚀 Funcionalidades

- Generación de partes médicos en PDF  
- Envío de reportes a familiares  
- Registro de información clínica de pacientes  --> solo la necesaria a la hora de mandar el parte
- Organización de partes por paciente  

## 🧩 Tech Stack

- Frontend: Vue.js
- Backend: Node.js
- Database: PostgreSQL
- Infraestructura: Docker & Docker Compose
